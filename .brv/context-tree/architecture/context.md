# Domain: architecture

## Purpose

Contains architecture and system design decisions for the AST grep MCP server project.

## Scope

Included in this domain:

- MCP transport choices
- Server scaffold decisions
- Agent-facing API surface
- Deployment considerations for local and remote use

Excluded from this domain:

- Implementation details of AST grep internals unless tied to server design
- General project notes unrelated to architecture

## Usage

Use this domain for documenting architecture decisions and transport choices for the MCP server.
